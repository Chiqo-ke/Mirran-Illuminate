# **App Name**: Mirran Architecture Portfolio

## Core Features:

- Project Showcase: Display projects with high-quality images and descriptions.
- Logo display: Showcase an aesthetic Mirran Architecture logo.
- About Us Section: Present the company's background, mission, and team.
- Contact Form: Contact form for inquiries and potential clients.
- Project Location Map: Interactive map showing project locations.

## Style Guidelines:

- Primary color: Dark navy blue (#2C3E50) for a modern and sophisticated feel.
- Background color: Dark gray (#34495E), providing a sleek backdrop to showcase the projects.
- Accent color: A muted gold (#95A5A6) to add a touch of elegance and highlight key elements.
- Font pairing: 'Montserrat' (sans-serif) for headlines to convey modernity, paired with 'Roboto' (sans-serif) for body text to ensure readability.
- Clean, minimalist icons related to architecture and design, using line-style icons in the accent color.
- Grid-based layout to provide structure, symmetry, and balanced content presentation.
- Subtle fade-in animations on project images when scrolling.
